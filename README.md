## 👩‍💻 Sobre mim

Sou desenvolvedora Full Stack com experiência na criação de aplicações **web e mobile**, atuando de ponta a ponta — da interface até a modelagem de dados e regras de negócio.

Minha trajetória começou com **PHP**, evoluiu para o ecossistema moderno com **React + TypeScript**, e hoje desenvolvo aplicações mobile com **React Native**, integradas a backends com **Supabase**.

💡 Foco em:
- Código limpo e escalável
- Arquitetura bem estruturada
- Experiência do usuário
- Soluções reais e eficientes
