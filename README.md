## Projetos

### [ShopSmart](https://github.com/KerollayneAkemy/shopsmart) · `React Native` `Expo` `TypeScript` `Supabase`

App mobile de gestão de listas de compras com autenticação segura e banco de dados relacional em produção.

- Autenticação completa via **Supabase Auth** (login, cadastro, logout, edição de perfil)
- Gestão de listas e produtos com controle de quantidade por unidade ou peso
- Barra de progresso dinâmica e cálculo automático de gastos por lista
- Banco de dados PostgreSQL com **Row Level Security (RLS)** — cada usuário acessa apenas seus próprios dados
- Estilização com **NativeWind/TailwindCSS** e ícones com Lucide

```
app/               # Rotas e navegação (Expo Router)
src/
├── components/    # Componentes reutilizáveis
├── hooks/         # Lógica de estado e efeitos
├── lib/           # Configuração do Supabase Client
├── types/         # Tipagens globais
└── utils/         # Funções auxiliares
```

> Projeto acadêmico em desenvolvimento — Fase 1/2 concluída. Desenvolvido em equipe.
